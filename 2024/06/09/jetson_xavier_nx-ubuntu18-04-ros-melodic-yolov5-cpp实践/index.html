rosжҚўжәҗ

еҺҹдёӯз§‘еӨ§жәҗеҫҲеӨҡеҢ…йғҪе·Із»ҸдёҚеҶҚиў«з»ҙжҠӨдәҶпјҢдҪҝз”Ёжё…еҚҺжәҗ

opencv

еӣ дёәjetson xavier nxиҮӘеёҰзҡ„opencvжІЎжңүж”ҜжҢҒcudaпјҢдёәдәҶиғҪеӨҹдҪҝз”ЁcudaпјҢйңҖиҰҒжәҗз Ғзј–иҜ‘opencv4.5гҖӮ

дҪҶжҳҜеңЁжҲ‘зҡ„жғ…еҶөдёӢпјҢжҲ‘зҡ„rosзҡ„е·ҘдҪңз©әй—ҙдёӯжңүеӨӘеӨҡзҡ„ж–Үд»¶е·Із»Ҹдҫқиө–иҮӘеёҰзҡ„opencvдәҶпјҢжүҖд»ҘдёҚиғҪзӣҙжҺҘеҲ йҷӨиҮӘеёҰзҡ„opencvиҖҢе®Ңе…ЁжӣҝжҚўдёәж–°зҡ„opencv4.5.4гҖӮеӣ жӯӨйңҖиҰҒиғҪеӨҹеҗҢж—¶е…Ғи®ёзі»з»ҹиҮӘеёҰopencvе’Ңж”ҜжҢҒcudaзҡ„opencv4.5.4

opencv4.5.4зј–иҜ‘

д»Һopencvе®ҳзҪ‘дёӢиҪҪopencv4.5.4зҡ„жәҗз ҒпјҢе’ҢеҜ№еә”зҡ„opencv_contribeзҡ„жәҗз ҒпјҢи§ЈеҺӢгҖӮ
дёәдәҶдёҚеҪұе“Қжң¬жқҘе°ұжңүзҡ„opencvпјҢеңЁ/usr/local/дёӢиҮӘе·ұж–°е»әдёҖдёӘopencv4.5.4зҡ„ж–Үд»¶еӨ№

иҝӣе…ҘеҲ°и§ЈеҺӢеҗҺзҡ„opencv-4.5.4ж–Үд»¶еӨ№дёӯпјҢиҮӘе·ұж–°е»әдёҖдёӘbuildж–Үд»¶еӨ№пјҢиҝӣе…Ҙе…¶дёӯпјҢдҪҝз”Ёд»ҘдёӢе‘Ҫд»ӨгҖӮ

е…¶дёӯпјҢCMAKE_INSTALL_PREFIXе°ұжҳҜеёҢжңӣжҠҠopencvжҢүиҪ¬еҲ°зҡ„зӣ®ж Үи·Ҝеҫ„пјҢOPENCV_EXTRA_MODULES_PATHжҳҜеҜ№еә”зҡ„opencv_contribдёӯзҡ„moduleж–Үд»¶еӨ№зҡ„зӣёеҜ№и·Ҝеҫ„гҖӮ

еңЁcmakeжҲҗеҠҹеҗҺпјҢе°ұеҸҜд»ҘmakeдәҶгҖӮ

еӣ дёәжҳҜеңЁjetsonжқҝеӯҗдёҠиҝӣиЎҢзј–иҜ‘пјҢжүҖд»ҘйңҖиҰҒзҡ„ж—¶й—ҙеҸҜиғҪдјҡйқһеёёй•ҝпјҢеҸҲдёҚжғіиҙ№зІҫеҠӣеҺ»еј„дәӨеҸүзј–иҜ‘дәҶпјҢжүҖд»ҘиҰҒе°ҪеҸҜиғҪзҡ„еҲ©з”ЁеӨҡж ёпјҢеңЁжҲ‘иҝҷйҮҢжҲ‘дҪҝз”Ёmake -j6еӣ дёәnxжңү6дёӘCPUж ёеҝғпјҢеҗҢж—¶ејҖдәҶдёҖдёӘtmuxзӘ—еҸЈпјҢиҝҷж ·еҸҜд»ҘйҳІжӯўдёҚе°ҸеҝғжҠҠmakeз»ҷжҢӮдәҶгҖӮ

еңЁmakeз»“жқҹеҗҺиҝҳйңҖиҰҒsudo make installпјҢжҠҠдёҖдәӣзӣёе…ізҡ„еә“ж”ҫеҲ°/usrзӯүзі»з»ҹзӣ®еҪ•дёӢгҖӮ

еҜ№еә”cv_bridgeзҡ„еӨ„зҗҶ

melodicиҮӘеёҰзҡ„cv_bridgeе’Ңopencv4.5дјҡжңүеҶІзӘҒпјҢжүҖд»ҘеҰӮжһңеёҢжңӣдҪҝз”Ёopencv4.5пјҢйңҖиҰҒиҮӘе·ұеңЁе·ҘдҪңз©әй—ҙдёӢдёӢиҪҪжәҗз Ғvision_opencvпјҢз„¶еҗҺдҪҝз”Ёе·ҘдҪңз©әй—ҙеҶ…зҡ„cv_bridgeгҖӮ

дёәдәҶиғҪеӨҹдҪҝз”Ёopencv4.5пјҢйңҖиҰҒдёӢиҪҪnoetic зүҲжң¬зҡ„vision_opencv,е№¶дё”дҝ®ж”№vision_opencv/cv_bridgeдёӢзҡ„CMakeLists.txt

дҪҶжҳҜд»…д»…дҝ®ж”№д»ҘеҗҺпјҢcmakeиҷҪз„¶еҸҜд»ҘдәҶпјҢдҪҶжҳҜеңЁзј–иҜ‘зҡ„ж—¶еҖҷд»Қз„¶дјҡйҒҮеҲ°жҠҘй”ҷвҖң
/home/nvidia/zal_ws/src/vision_opencv/cv_bridge/src/module.hpp: In function вҖҳvoid* do_numpy_import()вҖҷ:
/usr/include/python2.7/numpy/__multiarray_api.h:1537:144: error: return-statement with no value, in function returning вҖҳvoid*вҖҷ [-fpermissive]
#define import_array() {if (_import_array() < 0) {PyErr_Print(); PyErr_SetString(PyExc_ImportError, вҖңnumpy.core.multiarray failed to importвҖқ); return NUMPY_IMPORT_ARRAY_RETVAL; } }
вҖқ

ж №жҚ®жҠҘй”ҷдҝЎжҒҜпјҢзӣҙжҺҘеҺ»дҝ®ж”№жәҗз Ғпјҡ

иҝӣе…Ҙ/vision_opencv/cv_bridge/src/module.hppпјҢжҠҠdo_numpy_importеҮҪж•°зҡ„еЈ°жҳҺдҝ®ж”№дёә

еҶҚж¬Ўcatkin_makeпјҢжҲҗеҠҹзј–иҜ‘гҖӮ

еңЁе·ҘдҪңз©әй—ҙдёӯжҢҮе®ҡйңҖиҰҒзҡ„cv_bridgeе’Ңopencv

еңЁиҮӘе·ұзҡ„CMakeLists.txtдёӯжҳҺзЎ®жҢҮе®ҡйңҖиҰҒзҡ„opencvзүҲжң¬,cv_bridgeдёҚйңҖиҰҒдё“й—ЁжҢҮе®ҡпјҢcatkin_makeзҡ„ж—¶еҖҷдјҡдјҳе…ҲдҪҝз”Ёе·ҘдҪңз©әй—ҙеҶ…зҡ„cv_bridgeзҡ„жәҗз Ғзј–иҜ‘з”ҹжҲҗзҡ„еҢ…зҡ„гҖӮ

жҺҘдёӢжқҘпјҢеңЁд»Јз ҒдёӯпјҢе°ұеҸҜд»ҘжӯЈеёёзҡ„еҜје…Ҙopencv4.5е’Ңcv_bridgeдәҶгҖӮ
